## 🌐 Responsive Behavior

### Desktop (> 768px)
- Full table width with proper column widths
- Large, readable fonts
- Side-by-side signature areas
- Maximum width: 56rem (5xl)
- Padding: 32px-32px

### Tablet (640px - 768px)
- Optimized padding and spacing
- Medium font sizes
- Responsive grid layout
- Touch-friendly buttons (min 44px height)

### Mobile (< 640px)
- Compact padding: 16px
- Smaller fonts (still readable)
- Possibly stacked layout
- Full-width modal
- Touch-optimized buttons

## ✨ Animation Details

### Modal Open:
- Scale: 0.95 → 1
- Opacity: 0 → 1
- Duration: 300ms
- Easing: ease-out

### Condition Rows:
- Opacity: 0 → 1
- Staggered delay: 20ms per row
- Smooth transitions

### Hover Effects:
- Background: fade to light blue
- Duration: 200ms
- Z-index increase for subtle lift

---

## 📱 Mobile Optimization

### Touch Targets:
- Minimum 44px height for buttons
- Adequate spacing between interactive elements
- No hover effects on mobile (use active states)

### Text Readability:
- Minimum font size: 8px (for table)
- Line height: 1.2-1.4 (proper spacing)
- Good contrast ratios (WCAG AA)

### Viewport:
- Responsive viewport meta tag
- No horizontal scroll
- Proper padding on edges

## 🔄 Language Support

### Arabic (RTL):
- Direction: right-to-left
- Header: RTL text alignment
- Table: Column order preserved
- Borders: Mirrored for visual balance

### French (LTR):
- Direction: left-to-right
- Header: Normal text alignment
- Table: Standard layout
- Professional appearance

## 🎯 Design Principles Applied

1. **Minimalism**: Remove unnecessary elements
2. **Clarity**: Clear information hierarchy
3. **Efficiency**: Fit all content on one page
4. **Professionalism**: Business-appropriate design
5. **Consistency**: Unified design system
6. **Accessibility**: Proper contrast and readability
7. **Responsiveness**: Works on all devices
8. **Usability**: Intuitive interaction
