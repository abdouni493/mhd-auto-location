# 🖨️ CONDITIONS PRINT FEATURE - QUICK GUIDE

## 📋 Overview

The Conditions Print feature allows you to:
- ✅ Print rental conditions/terms with professional formatting
- ✅ Include client signature area on the left
- ✅ Include agency seal/cache on the right  
- ✅ Generate 2-page PDF ready for printing
- ✅ Fully customizable before printing

---

## 🚀 How to Use

### Option 1: From Contract Print Menu

1. **Open a Reservation**
   - Go to Planner page
   - Select a reservation

2. **Click Print Button** 
   - Click on reservation
   - Click "📄 Contrat" print button

3. **Choose Print Option**
   - Select "🖨️ Imprimer les Conditions" (Purple button)
   - This opens the Conditions Personalizer

4. **Print the Document**
   - Click "🖨️ Imprimer" button (bottom right)
   - Print dialog opens
   - Select printer and print

---

### Option 2: Customize Then Print

1. **From Contract Menu** → Click "📋 Personnaliser les Conditions"
   - Edit conditions if needed
   - Add/remove/edit conditions
   - Click "🖨️ Imprimer" when ready

2. **Print Dialog**
   - Review the print preview
   - Adjust printer settings if needed
   - Click "Print" in browser dialog

## 🎨 What You Can Customize

Before printing, you can:

✅ **Edit Conditions**
- Click condition to select it
- Click edit (pencil) icon
- Make changes
- Click "OK" to save

✅ **Add New Conditions**
- Click "+ Ajouter une Condition"
- Type new condition
- Click "Ajouter"

✅ **Delete Conditions**
- Select condition
- Click delete (trash) icon
- Removed from list

✅ **Reorder Conditions**
- Select condition
- Use up/down arrows
- Change order

✅ **Save to Database**
- Make your edits
- Click "Sauvegarder les conditions"
- Stored in database for future use

## 💡 Tips & Tricks

### **For Best Results**

1. **Page Setup**
   - Use A4 paper size
   - Portrait orientation
   - Normal margins

2. **Signature Space**
   - Signature area is 80px high
   - Good space for handwriting
   - Dashed box for seal/stamp

3. **Digital Signature**
   - Print blank document
   - Add signatures by hand
   - Scan for digital records

4. **PDF Save**
   - Choose "Save as PDF" instead of printer
   - Creates digital copy
   - Easy to store/send

## ❓ FAQ

**Q: Can I edit conditions after printing?**
A: Yes! Edit in the personalizer and print again. Each print will have the latest conditions.

**Q: Where should the client sign?**
A: On the left side in the large blank area with the line.

**Q: Where should the agency seal go?**
A: On the right side in the dashed box area on page 2.

**Q: Can I save as PDF?**
A: Yes! Choose "Save as PDF" in the print dialog.

**Q: Does it work on mobile?**
A: Yes, but recommend desktop/tablet for better print preview.

**Q: How many conditions can I add?**
A: Unlimited! They'll flow to multiple pages if needed.

**Q: Is it translated?**
A: Yes! Fully French/Arabic with RTL support.
