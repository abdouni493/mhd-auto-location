# Reports & Audit Interface - Enhanced Features

## Overview
The Reports & Audit page has been completely redesigned with comprehensive database integration and detailed analytics for all business metrics.

## Key Features Implemented

### 1. **Period Selection**
- Users can select custom start and end dates
- Click "Générer l'Audit Complet" button to generate reports for the selected period
- Date filtering is applied to all statistics and detailed reports

### 2. **Enhanced Statistics Dashboard**
The main overview now displays:
- **Revenus (Revenues)**: Total completed reservations revenue
  - Shows revenue in thousands (e.g., 150K DA)
  - Displays count of completed reservations
  
- **Dépenses (Expenses)**: Total store and vehicle expenses
  - Shows expenses in thousands
  - Breaks down store expenses separately
  
- **Profit Net (Net Profit)**: 
  - Calculates: Total Revenue - Total Expenses
  - Shows profit margin percentage
  - Dynamic color (green for profit, orange for loss)
  
- **Réservations**: Reservation statistics
  - Total reservations count
  - Active reservations count
  - Breakdown of pending/completed reservations
  
- **Kilométrage (Mileage)**: 
  - Total kilometers driven by all vehicles
  - Calculated from all reservations data

### 3. **Vehicle Performance Report (Car Statistics Table)**
Comprehensive per-vehicle analysis including:
- **Vehicle Name**: Brand, model, and registration
- **Réservations**: Total rental count
- **Revenus**: Total revenue generated by each vehicle
- **Entretien (Maintenance)**: Total maintenance costs incurred
- **KM**: Total kilometers driven
- **Dernier Usage (Last Use)**: Date of last rental
- **Dispo. (Availability)**: Availability percentage (100% if no active rentals)

### 4. **Client Performance Report (Client Statistics Table)**
Detailed client analysis with:
- **Client Name**: Full name of each client
- **Réservations**: Total number of rentals
- **Total Dépensé (Total Spent)**: Cumulative amount spent
- **Moyenne (Average)**: Average rental amount per reservation
- **Localisation (Location)**: Wilaya/region information
- **Dernier Rental (Last Rental)**: Date of most recent rental

### 5. **Reservation Summary**
Quick overview of reservation statuses:
- Completed reservations count
- Active reservations count
- Pending reservations count
- Average reservation price (highlighted in blue)

### 6. **Website Orders Report**
Displays all website orders with:
- Vehicle details
- Customer information
- Total price per order
- Order status and creation date
- Scrollable list for easy browsing

### 7. **Expenses & Maintenance Breakdown**
Detailed expense tracking:
- **Dépenses Magasin (Store Expenses)**: All store/general expenses
- **Dépenses Véhicules (Vehicle Expenses)**: Maintenance, vidange, assurance, contrôle
- Line-by-line itemization of all expenses
- Total calculations for each expense type

### 8. **Alerts Status Report**
Monitoring of vehicle maintenance alerts:
- Critical alerts count
- High-priority alerts count
- Detailed alert listing with:
  - Alert title
  - Days until expiration
  - Alert severity level
- Color-coded by severity (red for critical, orange for high)

## Data Calculations

### Revenue Calculations
```
Total Revenue = Sum of completed reservations total_price
Average Reservation Price = Total Revenue / Number of completed reservations
```

### Expense Calculations
```
Store Expenses = Sum of all store_expenses costs
Vehicle Expenses = Sum of all vehicle_expenses costs
Total Expenses = Store Expenses + Vehicle Expenses
```

### Profit Calculations
```
Net Profit = Total Revenue - Total Expenses
Profit Margin = (Net Profit / Total Revenue) * 100%
```

### Vehicle Statistics
```
Total KM Driven = Sum of (kmAtReturn - kmAtDeparture) for all reservations
Availability = 100% if no active reservations, 75% if active
```

### Client Statistics
```
Total Spent = Sum of all reservation prices for each client
Average Rental = Total Spent / Number of Reservations
Last Rental = Most recent reservation date
```

## Database Connections
The page connects to the following database tables:
- `clients` - Client information
- `reservations` - Reservation data for revenue and analysis
- `cars` - Vehicle information
- `workers` - Employee data
- `store_expenses` - General business expenses
- `vehicle_expenses` - Vehicle maintenance and care expenses
- `maintenance_alerts` - Vehicle maintenance alerts
- `website_orders` - Online orders data
