## 🎯 Key Features

### ✨ Modern Design Elements
- **Gradient Header**: Blue gradient background (from-blue-600 to-blue-700)
- **Clean Typography**: Clear hierarchy with semibold/bold weights
- **Professional Colors**: Blue accents on white background
- **Smooth Animations**: Fade-in effects on rows
- **Hover Effects**: Subtle blue background on row hover

### 📱 Responsive Breakdown
- **Mobile**: Compact padding, stacked layout, touch-friendly
- **Tablet**: Optimized spacing, 2-column signature grid
- **Desktop**: Full-width table, maximum content visibility

### 🌍 Bilingual Support
- **Arabic (RTL)**: Right-to-left text direction, mirrored layout
- **French (LTR)**: Left-to-right standard layout
- **Toggle Buttons**: Easy language switching

### 🖨️ Print Optimization
- **Single A4 Page**: All 15 conditions fit perfectly
- **Professional Quality**: Optimized fonts and spacing
- **Print CSS**: Proper color preservation and margins
- **Both Languages**: Print in selected language

```

    ↓

    ↓

    ↓
Click "Imprimer les Conditions" or "طباعة الشروط"
    ↓
Conditions Modal Opens
├─ Gradient header with title
├─ Language toggle (Arabic/French)
├─ Professional table layout
├─ All 15 conditions visible
├─ Signature preview
└─ Action buttons
    ↓
Switch Language (if needed)
    ↓

    ↓

    ↓
```
